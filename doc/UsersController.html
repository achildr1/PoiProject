- (Object) create

Create a new User that can log into this application.

Parameters:

  • user_name (String) —

    the unique name used to identify the corresponding user in this system

  • first_name (String) —

    the first name of the person for whom this User is being created for

  • last_name (String) —

    the last name of the person for whom this User is being created for

  • password (String) —

    the secret text that will be used to authenticate the corresponding User in this system

  • password_confirmation (String) —

    a duplicate entry of the given password used to confirm that no erraneous characters were entered in the password

  • email (String) —

    the address used by this system in the event that the User cannot remember her password

  • the (Role) —

    Role object that should be associated with this User

Preconditions:

  • The given user_name is a unique record and the first_name, last_name, password, and email are defined.

Postconditions:

  • A new User with the specified user_name, first_name, last_name, password, and email is added to the database.

def create
	@user.role = Role.find_by(name: "Guest") if @user.role.nil?
	if @user.save
		flash[:notice] = "Your account has been successfully created!"
		UserMailer.welcome_email(@user).deliver
		redirect_to root_url
	else
		flash[:error] = get_all_errors
		render 'new'
	end
end

def destroy
	if @user == current_user
		session[:user_id] = nil
		@current_ability = nil 
	end
	@user.destroy
	flash[:notice] = "User was successfully deleted."
	redirect_to root_url
end

def update

		flash[:notice] = "User account was updated successfully."
		redirect_to user_path(@user)
	else
		flash[:error] = get_all_errors
		render 'edit'
	end
