- (Object) create

Add a new Comment to the database with the user-specified author and body. The user will be redirected to the calling object.

Parameters:

  • author (String) —

    the name or alias of the person who wrote the comment

  • body (String) —

    the comment text

  • commentable (Commentable) —

    the object the comment is to be created for

Preconditions:

  • A new empty Comment should exist before this method is called.

Postconditions:

  • A new Comment will be added to the database with the given attributes. Calling Comments.size? after this method has executed successfully will return one more than the number returned had the method been called before executing this method. If the author or body is not defined when this method executes, then an error message will be displayed.

def create
	@comment.commentable = @commentable		## @commentable is the object passed to this controller in the partial, whether it's an interest point or whatever; so this line just sets the association
	if @comment.save
		flash[:notice] = "Thanks for the comment!"
		redirect_to @commentable
	else
		flash[:error] = "Sorry! Monkeys just ate your comment. Please try again."
		redirect_to @commentable
	end
end
